网易笔试题-组合和

组合之和程序题

题目描述

某直播平台内，用户可以花费游戏币给主播赠送礼物。每种礼物都有一定的价格，主播收到礼物后可以增加一定的魅力值。比如紫水晶4900游戏币/个，主播收到后可以增加1000点魅力值/个；幸运彩虹1000游戏币/个，主播收到后可以增加100点魅力值/个。
现在有个土豪要给主播送礼物，主播只让土豪一次送一个礼物，而且要求土豪至少送出若干次礼物。土豪希望在满足主播要求的前提下，尽快花完游戏币，而且希望尽可能多的为主播增加魅力值。
请你帮土豪算一下，最少送多少次礼物可以刚好把游戏币都花完，此时能给主播增加多少魅力值。如果这个次数存在多种送礼方案，求出能让主播增加最大魅力值的方案。

输入

第1行是整数n，表示土豪拥有的游戏币数量，0 < n < 5000。
第2行是整数l，表示主播要求土豪至少送出多少次礼物，0 < l <= 30。
第3行是整数m，表示共有m种礼物，0 < m < 300。
从第4行开始有m行，每行是两个整数c和p，c表示该礼物价格，p表示主播能增加的魅力值，0 < c <= 1000，0 < p <=1000。

输出

整数t和整数q，中间用空格分开，t表示土豪最少送t次礼物可以刚好把游戏币花完，q表示送t次礼物把游戏币花完的前提下，主播最多可以增加q点魅力值。
如果输入的礼物组合不能刚好花完游戏币或者达不到主播的要求的最少次数，输出-1 -1。

样例输入

10000
3
5
9999 15000
1 10
1000 500
2500 1000
7000 10000

样例输出

4 11500
